==================================
Pengiriman pesan via python
==================================

Cara menggunakan:
 - Edit kontak.txt dengan format 
	tujuan;Receiver 1
	untuk tujuan, bisa hanya 4 nomor belakang
	contoh : - 1853;Receiver 1
- ketik python bot.py di terminal atau git bash

untuk proram ini membutuhkan:

1. Selenium
	Cara install selenium :
		pip install selenium
2. Chromedriver
3  Chrome browser
